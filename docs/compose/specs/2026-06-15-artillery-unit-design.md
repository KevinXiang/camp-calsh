# 火炮兵设计方案

## [S1] 定位

**攻城 + 溅射远程单位** — 超远程轰击军营，命中后产生大范围溅射伤害。需要前排保护，是推进和拆营的核心输出。

### 设计目标

1. 提供专门的攻城手段，加速破营节奏
2. 大范围溅射惩罚密集站位
3. 脆皮慢速，需要前排保护
4. 抛物线弹道 + 火焰爆炸特效，视觉辨识度高

## [S3] 弹道表现

- **抛物线弹道**：炮弹向上弧线飞行，落地后爆炸
- 弹道速度：180 px/s
- 弹道生命周期：2.5s
- 命中判定距离：12px

### 抛物线实现

使用现有 Projectile 系统，增加 `arcHeight` 属性控制弧线高度。渲染时根据飞行进度计算 Y 轴偏移：

```
renderY = baseY - arcHeight * sin(π * elapsed / maxTime)
```

## [S4] 战斗机制

### 溅射伤害

1. 炮弹命中目标后，在命中点 80px 圆范围内对所有敌方单位和军营造成伤害
2. 溅射伤害 = 基础攻击（12），不享受攻城倍率
3. 主目标享受攻城倍率（2x）

### 攻城加成

1. 对军营伤害 = 基础攻击 × 2 = 24
2. 对小兵伤害 = 基础攻击 = 12
3. 溅射范围内对军营也享受 2x

### 无最小射程

任何距离都可开火，被近身仍能攻击。

## [S5] 特效设计

### 飞行阶段

- 炮弹：深灰色圆球 + 橙色火焰尾部
- 尾迹：灰色烟雾粒子，渐隐消失

### 命中爆炸（火焰风格）

1. **火焰核心**：橙红色圆形扩散（0.3s）
2. **烟圈**：灰色圆环向外扩散（0.5s）
3. **冲击波**：白色半透明圆环快速扩散（0.2s）
4. **碎片飞溅**：4-6 个小碎片向四周弹射
5. **地面焦痕**：命中点留下短暂焦黑痕迹（1s 后消失）

### 溅射效果

- 溅射范围内每个被命中单位产生**火花闪烁**（区别于普通 hit 特效）
