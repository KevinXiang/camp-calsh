# 军营大作战 (Camp Clash)

一款 Web 端 2.5D 放置对战沙盒模拟器。玩家可以在无限画布上自由放置红蓝军营，军营自动生产 Q 版玩具小兵，小兵自动寻敌并进行低暴力对战。

## 核心玩法

- 自由放置红蓝双方军营
- 军营自动生产 7 种兵种小兵
- 小兵自动寻敌、移动、攻击
- 升级军营提升产兵速度、生命值、攻击力
- 暂停/加速/清场，观察不同战场配置的对抗结果

### 升级系统

每个军营可独立升级 3 类属性：

- **产兵升级** - 降低产兵间隔（最高 30%）
- **生命升级** - 提升军营和小兵生命值（最高 +30%）
- **武器升级** - 提升小兵攻击力（最高 +30%）

## 架构设计

### 固定步长模拟

`SimulationClock` 将变帧率切分为固定 1/60s 步长，确保逻辑一致性。`speed` 倍率只影响每帧产出的步数。

### 事件驱动 UI

`UiBridge` 使用事件发射器模式，游戏层和 UI 层通过它通知状态变化（placementChanged / selectionChanged / simChanged / statsChanged / gameOver）。

### 黑盒状态

所有游戏逻辑共享一个 `GameState` 实例，直接读写 `camps/units/projectiles/events` Map。
